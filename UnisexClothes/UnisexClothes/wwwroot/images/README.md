## Lưu ý:
1. Tất cả hình ảnh nên là file JPG hoặc PNG
2. Kích thước đề xuất tối ưu cho web (không quá 200KB/file)
3. Nên có phiên bản retina (2x) cho màn hình độ phân giải cao
4. Hình ảnh sản phẩm nên có nền trắng hoặc trong suốt

## Nguồn ảnh miễn phí bạn có thể sử dụng:
- Unsplash (https://unsplash.com) - Tìm kiếm "minimalist fashion", "unisex clothing"
- Pexels (https://pexels.com) - Tìm kiếm "clothing", "fashion"
- Pixabay (https://pixabay.com) - Ảnh miễn phí cho thương mại
